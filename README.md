
# Social Media API

Basic Social Media API that implements the following:

* Create Posts alongside some pictures or videos

* Make Comments on Posts (including pictures or videos)

* Freely Interact with Posts and Comments with multiple reactions



## Installation

* Download and unzip or clone the repo 
```bash
  git clone https://gitlab.com/axtrosoft/training/laravel/mhd-social-media.git
```

* Update with Composer
```bash
  composer update
```
* Require the Spatie-media-library 
```bash
  composer require "spatie/laravel-medialibrary:^10.0.0"
```
* Create DB and Run Migrations 
```bash
  php artisan migrate
```
* Run the app !
```bash
  php artisan serve
```
And You're Good To Go
